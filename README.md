# TimeLine
Dieses Tool ist ein KI-generiertes Studienprojekt. Der gesamte Code wurde per Prompt-Engineering erstellt, um zu zeigen, wie Fachkräfte ohne Coding-Skills eigene Tools für die Biografiearbeit bauen können. Feature: Wandelt Transkripte via Mistral-KI in Zeitstrahlen. Nutzung: Nur mit eigenem API-Key.
